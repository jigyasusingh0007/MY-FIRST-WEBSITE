# Akash Singh — Portfolio

## Files
- index.html      -> the page structure (HTML)
- css/style.css   -> all the styling
- js/script.js    -> theme toggle, typing animation, smooth scroll
- images/akash.jpg -> your photo

## How to run in VS Code
1. Open VS Code -> File -> Open Folder -> select this "akash-portfolio" folder.
2. Easiest: right-click index.html in your file explorer -> Open with Chrome.
3. Recommended (auto-refresh): install the "Live Server" extension in VS Code,
   then right-click index.html -> "Open with Live Server".

## To edit
- Change text: open index.html
- Change colors/spacing: open css/style.css (top me --accent color hai)
- Change typing roles / behaviour: open js/script.js
- Change photo: replace images/akash.jpg with your own (keep the same name)
- CV: Akash_Singh_Resume.pdf project me add ho chuka hai, Download CV button uspe link hai. Replace karna ho to wahi naam rakhna.
